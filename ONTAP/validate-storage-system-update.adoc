= ONTAP 스토리지 시스템 업데이트 상태를 확인합니다
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
BlueXP  소프트웨어 업데이트를 사용하여 ONTAP 업데이트 상태를 확인할 수 있습니다.

.단계
. BlueXP  왼쪽 탐색 모음에서 * 상태 > 소프트웨어 업데이트 * 를 선택합니다.
. ONTAP 업데이트 상태를 보려면 * 기록 * 탭을 클릭하십시오.



NOTE: 클러스터가 기록 탭에 반영되려면 2~3시간이 걸립니다.
